## getting started

> [!CAUTION]
> tested only on macos

### prerequisites

- rust
```sh
 # development phase started by using this specific version, any later toolchain must be fine to use
rustup toolchain install 1.94.0
```

### usage

```sh
[dependencies]
dene = { git = "https://codeberg.org/lolp1ke/dene.git" }
```
